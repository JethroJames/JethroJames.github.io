# VidPair-Halluc Project Page

Static project page for the VidPair-Halluc benchmark.

- Live page: https://jethrojames.github.io/VidPair-Halluc/
- Dataset: https://huggingface.co/datasets/Jethro37/VidPair-Halluc
- Code: https://github.com/JethroJames/VidPair-Halluc

## Preview

```bash
python -m http.server 8794
```

Then open `http://127.0.0.1:8794`.

## GitHub Pages

Keep the repository private during review. Do not enable GitHub Pages until the
authors approve the page for public release. When approved, publish from the
repository root on the `main` branch. The page is fully static and does not need
a build step.

## Contents

- `index.html`: project page
- `styles.css`: responsive styling
- `assets/videos/`: web-friendly highlight videos from the dataset
